# 📊 Relatório Interativo com Parâmetros

## 🎯 Objetivo

Criar uma análise dinâmica no Power BI usando **parâmetros de campo**, permitindo alternar entre diferentes visões de dados.

---

## 🧩 Estrutura

* **Visão 1 (Categorias):** análise por dimensões como categoria/produto
* **Visão 2 (Valores):** análise por métricas como vendas, lucro e descontos

---

## 📖 Storytelling

* Identificar **onde estão as vendas**
* Comparar **indicadores (sales vs profit)**
* Detectar **oportunidades (alto volume x baixa rentabilidade)**

---

## 🚀 Conclusão

Relatório dinâmico, interativo e focado em apoiar a tomada de decisão.
